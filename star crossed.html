The purpose of this project was to give efficient advice to high school students. It would help with giving sound advice on both socializing with friends and romantic relationships. The app had a main page which talked about the different compatibilities with a specific zodiac sign giving ideas on whether that zodiac sign has traits that work well with yours. There is also a chatbot which was trained to aid with anything related to zodiac signs and relationships. This was the objective of the application. My role in this project was the project manager. I was responsible for overseeing the progress of the project. I also tasked the other members of the team of what needs to be done. I learned about bootstrap, which is used for web development on the CSS end. I also learned about time management since web designing and leading a team take a lot of time and effort. I also learned a lot about the various design elements such as navbars and animated elements to web designing. Some major challenges was that our initial feature that we really wanted to include was the two-way chat feature. It would work how iMessages or WhatsApp works. Sadly, we struggled with make it so that you people could chat on the same chat and interact with it. To overcome this, we opted for the chatbot instead so that our app would still function and do what is supposed to to some extent. My teamwork skills were demonstrated by how efficiently our team worked together, especially on the last day, to fix up any last bit of problems and details we wanted to fix. We communicated very effectively and created a project that was pretty good even though we hit a couple rough patches.
