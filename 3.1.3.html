The purpose of this project was to create a online ordering system in which the user to pick and chose what they want to order and then the price would be compiled for them. It was made for anyone who wishes to buy food. In this project, I was the project lead. I learned many things as python, at the time, was relatively new to me. I learned how to connect different parts of programs together and work with this long piece of code. This led to some challenges as sometimes the code would not work, and fixing it was a bit hard for me. However, with the assistance of my peers and various other resources, I was able to fix those problems. Since this was a solo project, there no teamwork done but in a way I showed “leadership” as I owned up the project and worked efficiently.
